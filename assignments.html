Assignment 10: Image Retrieval

Assignment 10 is focused on developing a simplified version of Google Image Search. This GUI-based project involves creating an application where users can input a text or image query and retrieve the top 5 relevant images from a database, displaying them along with their similarity scores. The interface allows users to search by text, image, or a hybrid mode, offering control over the relevance of text versus image through adjustable weighting. The project provides a practical application of image and text embedding technologies in a user-friendly search engine.
